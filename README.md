# 🌤️ Real-Time Weather Dashboard

A real-time weather dashboard built using Python, Streamlit, and OpenWeatherMap API.

## Features

- Live Weather Data
- City Search
- Temperature
- Humidity
- Wind Speed
- Pressure
- Visibility
- Weather Alerts
- Auto Refresh

## Technologies Used

- Python
- Streamlit
- Requests
- OpenWeatherMap API

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
